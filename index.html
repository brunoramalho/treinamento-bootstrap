O que é e porque usar Layout Responsivo?


Responsividade tem crescido junto com o crescimento do mercado de dispositivos móveis, principalmente por conta de tablets e celulares de tamanhos e resoluções cada vez mais variadas.


Neste novo contexto de várias resoluções, responsividade surge como uma evolução lógica do desenvolvimento web.


Claro que devemos ter em mente que responsividade não resolve todos os problemas e talvez nem seja a solução definitiva para todos, mas com certeza é um caminho a ser seguido e ainda mais explorado.

Fixo x Líquido x Responsivo


Este é o formato que mais utilizamos, talvez o mais comum, por enquanto. Neste modelo a página tem tamanho fixo para tudo, independente de onde está sendo visualizada.

Fixo x Líquido x Responsivo


Este tipo de design se preocupa em construir tudo em tamanhos flexíveis (%, em, rem), nunca, ou quase nunca utilizando números fixos na sua construção. O tamanho do site será, por exemplo, 80% da tela, a sidebar terá 30% do e o conteúdo terá 70% todos relacionados aos 80% da tela.

Fixo x Líquido x Responsivo


Nesta técnica o site é construído de forma que ele tente se adaptar o melhor possível a tela em que esta sendo carregado. A idéia aqui é proporcionar a melhor experiência para o usuário. Essa técnica utiliza bastante das media queries para definir como o site deve se comportar em cada momento, é uma mistura do design fixo e design fluído.

Media Queries


As Media Queries definem condições para a utilização de um CSS específico. Se essas condições forem aprovadas, ou seja, se o dispositivo de adequar a todas as condições, o CSS será aplicado.



body { background: blue; }

@media screen and ( max-width: 320px ) {
   body { background: red; }
}

@media screen and ( min-width: 321px ) and ( max-width: 480px ) {
   body { background: green; }
}
